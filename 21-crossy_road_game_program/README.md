# PONG GAME

A simple Python game built with the turtle module, inspired by the classic Frogger.
The goal is to help the turtle cross the screen while avoiding cars. Each successful crossing increases the level and the car speed.
---

## Features
- Player moves upward using the **Up Arrow** key.  
- Randomly generated enemies with different colors.  
- Level and speed increase after each crossing.  
- Displays **Game Over** message when the player collides with an enemy.
