## Description

Calculates a "love score" between two names:

- Combine the names  
- Count letters in **TRUE**  
- Count letters in **LOVE**  
- Combine totals to form a 2-digit score  
- Print the final score  

### Example

```python
calculate_love_score("Kanye West", "Kim Kardashian")
# Output: 42
```
