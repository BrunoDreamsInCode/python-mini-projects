# Pong Game - Python Turtle

A simple Pong game implemented in Python using the Turtle graphics library.  
This project is designed for learning Python OOP (classes, methods) and basic game development concepts.

---

## Features

- Two-player game (left and right paddles)
- Ball movement with collision detection
- Scoreboard for each player
- Increasing ball speed after each paddle hit
- Simple and clean code structure for study purposes

---

## Controls

- **Right Paddle:**  
  - Move Up: `Up Arrow`  
  - Move Down: `Down Arrow`

- **Left Paddle:**  
  - Move Up: `W`  
  - Move Down: `S`

---

## Installation & Run

Navigate to the project folder:

```bash
    cd pong-python
    python main.py
```
